# SQL-Lanches

O contexto da Entrega é a SQLanches, uma lancheria que vende hambúrgueres, acompanhamentos e bebidas.

As entidades (tabelas) que fazem parte desse contexto são as seguintes:

Clientes
Endereços
Produtos
Pedidos
Produtos_Pedidos (tabela pivô)
Diagrama ER
